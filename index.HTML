IMDB Search Improvement



This project aims to improve the search functionality on IMDB by utilizing a web crawler to gather additional information about movies and actors



Features



  • Collaborations of actors in movies: The crawler gathers information about the movies in which a given actor has appeared and the other actors with whom they have collaborated

  • Search by start and end time of movie: The crawler gathers information about the release date of movies and allows users to search for movies within a specific date range

  • Search by actor and start and end of user rating: The crawler gathers information about the user ratings for movies and allows users to search for movies by a specific actor within a range of user ratings
